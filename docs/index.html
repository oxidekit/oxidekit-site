Build Native Desktop Apps
Without the Bloat

OxideKit is a Rust-native application platform. No browser, no JavaScript, no bundlers. Just fast, native apps.

Get Started Free View on GitHub
cargo install oxide-cli && oxide new my-app

Why OxideKit?

Native Performance

GPU-accelerated rendering with wgpu. Your apps start instantly and run smoothly. No browser overhead, no JavaScript VM.

Hot Reload

State-preserving hot reload. Change your UI and see results instantly without losing application state.

Secure by Default

Built-in permission system and network allowlists. Your users know exactly what your app can access.

Declarative UI

Clean .oui syntax for describing your UI. Components, layouts, and styling in one place.

Extension Ecosystem

Plugins for UI components, native capabilities, and themes. Install with oxide add.

Cross-Platform

Build once, deploy to macOS, Windows, Linux. Mobile support coming soon.
